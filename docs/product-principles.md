# 무슨일 제품 원칙

무슨일은 집회·시위 정보를 객관적으로 정리해 시민의 알권리를 돕는 서비스다.

무슨일은 참여 독려, 찬반 여론, 댓글 토론, 불법/합법 단정, 충돌 전술 안내 서비스가 아니다.

## 공개 화면 원칙

모든 주요 공개 화면은 다음 네 가지를 기준으로 구성한다.

- 확인된 내용
- 아직 확인 중
- 다른 주장
- 확인 정도

## 비공개 원칙

- 사용자 원문을 그대로 공개하지 않는다.
- 정밀 위치, 얼굴, 차량번호, 주소, 전화번호, 미성년자 식별 정보는 공개하지 않는다.
- 신고 수, 후원, 조회수, 공유 수는 노출 순서와 신뢰 판단에 쓰지 않는다.

## 판단 원칙

- 경찰, 정부, 언론, 주최 측, 시민, AI 중 어느 쪽도 자동으로 진실이 아니다.
- 공식 자료도 자동 결론이 아니라 출처가 명확한 자료로 표시한다.
- 확인되지 않은 내용은 확정 표현 없이 표시한다.
